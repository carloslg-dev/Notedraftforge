# NoteDraftForge — Project Context

## Purpose
Private-first Markdown-based editor for writing and interpreting poems, songs, and structured text.

## MVP Goals
- Usable for real writing
- Support annotations and layers
- Keep content and interpretation separated

## Non-Goals
- No AI
- No backend
- No collaboration

## Core Concepts
- Piece
- Annotation
- Layer
- Markdown as source of truth

## Architecture
- Lightweight DDD
- Hexagonal (pragmatic)

## Workflow
- SDD
- GitHub issues required
